# rekstor

person record store application with CLI and API

## Usage

Print Help:
```bash
$ java -jar rekstor.jar
```

CLI Usage: 
```bash
$ java -cp rekstor.jar rekstor.core [<file> \"<delimited-pattern>\"]
```

API Usage:
```bash
$ PORT=<port> java -cp rekstor.jar rekstor.handler
```
